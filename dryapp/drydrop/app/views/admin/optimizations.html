Javascriptmore info

These are optimizations for your Javascript files.

Gzip compression reduces response times and saves up to 70% of bandwidth by losslessly reducing the size of files. It is generally advisable to use gzip compression on all HTML, Javascript, and CSS files.

The "never expire" option sets a far future Expires header so that cached files will stay cached indefinitely. This is useful for content which is not expected to change. However, if the content needs to be changed then the name of the file will also need to be changed. This is good for libraries, but not for content you will be updating. This option is appropriate for deployment sites, but not for development.

Minification of Javascript rewrites Javascript code to be smaller than is possible with just gzip compression, giving up to 60% savings in terms of size.
